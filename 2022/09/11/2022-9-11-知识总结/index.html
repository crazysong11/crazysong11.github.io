什么是 Laravel ？

1. Laravel

Laravel是一个基于 PHP 语言的 Web 开源框架，采用了 MVC 模式[2]，由于 Laravel 具备 Rails[3] 敏捷开发等优秀特质，深度集成 PHP 强大的扩展包（Composer）[4]生态与 PHP 开发者广大的受众群，让 Laravel 在发布之后的短短几年时间得到了极其迅猛的发展。

2. MVC 模式

一种软件架构模式，它将软件分为以下三个部分：
Model ：模型
View : 视图
Controller ：控制器

3. Rails

全称叫 Ruby on Rails ，是一个基于 Ruby 语言的 Web 开源框架，强调和注重敏捷开发。

4. PHP Composer
